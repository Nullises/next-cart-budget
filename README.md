# next-cart-budget

Next Cart with budget logic

## Run Backend

1. `cd backend`
2. `npm install`
3. `npm run dev`

### Backend online

`https://next-cart-budget.vercel.app`

## Run Frontend

1. `cd frontend`
2. `npm install`
3. Create .env file with current BACKEND API URL (show .env.template)
4. `npm run dev`

### Frontend online

`https://next-cart-budget-frontend.vercel.app/`
